Notable Projects

Open ID Connect

Recently at my job, I, along with my team, implmented OAuth2 and Open ID Connect into our company websites. This project was initially estimated at about 2 weeks of work, oh how quickly did we realize our error. As anyone who has dealt with authentication might know, these things are a can of worms. First it was client secrets not getting encrypted correctly, then it was roles not getting attached to the access token, then it was the refresh token not doing anything when the access token expired, then it was...

But in the end I got it figured out. My key takeaways from this project are twofold. The first being that AI is as wrong as often as it is right. During this whole process, I was using AI extensively to help me debug and troubleshoot (my company wanted us to experiment with it and see if it can help us improve development), however I am getting the inkling that it was making me move slower. Debugging not only my own issues, but also the ones the AI was introducing. My second takeaway is that I need to leverage my coworkers more. Even if they are not experts in the subject matter, they can help me more than the AI can. Near the end of the project we discovered that a separate department in the company had already implemented an older version of OIDC, and using their code we were able to complete the last phase of the project faster than anticipated.

In the end, I think I came away from this project a better developer. Not only was I able to improve my general skills, I was also able to learn more about website authentication. I now assist my coworkers in their new websites using our new shared authentication.
